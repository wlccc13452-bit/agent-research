# 前端缓存清除指南

## 问题：前端显示"暂无K线数据"

**原因**：React Query缓存了旧的空数据

## 解决方案（按推荐顺序）

### 方案1：硬刷新浏览器（最简单）✅

```
Windows: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

### 方案2：清除浏览器存储（最彻底）✅

**步骤**：
1. 打开浏览器开发者工具（F12）
2. 切换到 **Application** 标签
3. 左侧找到 **Storage** → **Local Storage**
4. 右键点击 `http://localhost:5173`
5. 选择 **Clear**
6. 同样清除 **Session Storage**
7. 刷新页面（F5）

### 方案3：使用无痕模式（最干净）✅

```
Windows: Ctrl + Shift + N
Mac: Cmd + Shift + N
```

然后访问 `http://localhost:5173`

### 方案4：清除所有站点数据

**Chrome浏览器**：
1. 按 F12 打开开发者工具
2. 切换到 **Application** 标签
3. 左侧找到 **Storage**
4. 点击 **Clear site data** 按钮
5. 刷新页面

**或者使用浏览器菜单**：
1. 点击浏览器地址栏左侧的锁图标
2. 点击 **Site settings**
3. 点击 **Clear data**
4. 刷新页面

### 方案5：禁用缓存后刷新

**步骤**：
1. 按 F12 打开开发者工具
2. 切换到 **Network** 标签
3. 勾选 **Disable cache** 复选框
4. 保持开发者工具打开状态
5. 刷新页面（F5）

## 为什么会出现这个问题？

1. **React Query缓存机制**：
   - 默认缓存时间：5分钟
   - 默认过期时间：30秒
   - 如果在过期时间内重复请求，会返回缓存数据

2. **历史原因**：
   - 之前数据库为空时，前端请求过空数据
   - React Query缓存了这个空结果
   - 现在数据库有数据了，但缓存还是旧的

## 验证是否成功

清除缓存后，应该看到：
- ✅ K线图表正常显示
- ✅ 显示100条K线数据
- ✅ 数据日期范围：2025-09-30 ~ 2026-03-06
- ✅ 平均加载时间 < 50ms

## 预防措施

### 1. 添加缓存版本控制

在 `frontend/src/services/api.ts` 中：

```typescript
const CACHE_VERSION = 'v1';

const queryClient = new QueryClient({
  defaultOptions: {
    queries: {
      staleTime: 30 * 1000,
      cacheTime: 5 * 60 * 1000,
      queryKeyHashFn: (key) => {
        return `${CACHE_VERSION}-${hashQueryKey(key)}`;
      },
    },
  },
});
```

### 2. 添加开发环境自动清除缓存

在 `frontend/src/main.tsx` 中：

```typescript
if (import.meta.env.DEV) {
  // 开发环境下，启动时清除旧缓存
  const queryClient = new QueryClient({
    defaultOptions: {
      queries: {
        staleTime: 0, // 开发环境立即过期
      },
    },
  });
}
```

### 3. 添加手动刷新按钮

在 K线图表组件中添加刷新按钮：

```tsx
import { useQueryClient } from '@tanstack/react-query';

function KLineChart() {
  const queryClient = useQueryClient();

  const handleRefresh = () => {
    queryClient.invalidateQueries(['kline-quick']);
    queryClient.invalidateQueries(['kline-full']);
  };

  return (
    <div>
      <button onClick={handleRefresh}>刷新数据</button>
      {/* K线图表 */}
    </div>
  );
}
```
