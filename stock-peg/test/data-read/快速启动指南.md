```
双击 "启动测试工具.bat"
```

## 📋 功能选项

### [1] GUI界面模式（推荐）

图形界面，支持：
- 下拉选择股票代码
- 切换数据类型（实时行情、K线、报告等）
- 查看JSON格式数据
- 一键复制到剪贴板

### [2] 命令行测试模式

快速验证：
- 数据库连接
- 股票列表
- 行情数据
- K线数据
- 每日报告

## 🔧 技术特性

✅ **完全复用Server端代码**
- 使用相同的数据库连接
- 使用相同的服务类
- 使用相同的数据模型

✅ **不通过WebSocket**
- 直接读取数据库
- 无需启动Server
- 本地快速测试

✅ **完整的数据支持**
- 实时行情
- K线数据
- 每日报告
- 基本面指标
- 数据来源追踪

## 📝 注意事项

1. **数据库位置**: `backend/data/stock_peg.db`
2. **依赖要求**: `pip install sqlalchemy aiosqlite`
3. **编码问题**: Windows控制台可能需要UTF-8编码
