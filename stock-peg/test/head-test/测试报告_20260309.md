## 解决方案

### 立即操作

1. **修复自持股票文件**:
   ```bash
   # 修改 d:/play-ground/股票研究/stock-peg/自持股票.md
   - 神华股份 → 中国神华
   - 赣锋理业 → 赣锋锂业
   ```

2. **获取香农芯创数据**:
   ```bash
   # 启动后端服务，自动获取
   cd backend
   python main.py
   
   # 或手动调用API
   curl -X GET http://localhost:8000/api/stocks/quote/香农芯创
   ```

### 补充指数数据

1. **A股指数**:
   ```bash
   # 更新深证指数
   curl -X POST http://localhost:8000/api/stocks/update/399001/day
   
   # 更新创业板指
   curl -X POST http://localhost:8000/api/stocks/update/399006/day
   ```

2. **美股指数**:
   - 需要先配置美股数据源（在 `backend/config/settings.py` 中配置）
   - 支持的数据源：Alpha Vantage / Finnhub / Yahoo Finance

### 使用方法

```bash

# 方法2：直接运行Python脚本

  d:/play-ground/股票研究/stock-peg/test/head-test/check_missing_data_v2.py
```

### 核心原理

测试程序完全依赖server数据库中的股票代码映射，不使用任何硬编码：

1. **从 StockRealtimeQuote 表读取**: server通过外部API获取股票行情时保存的数据
2. **建立映射关系**: stock_name → stock_code
3. **检查数据完整性**: K线数据 + 财务数据
