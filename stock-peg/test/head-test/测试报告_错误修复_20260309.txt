问题1: technical_indicators_cache 导入错误
--------------------------------------------------------------------------------
位置: backend/services/stock_service.py 第468行
错误: cannot import name 'technical_indicators_cache' from 'services.extended_cache'

原因:
- stock_service.py 尝试导入 'technical_indicators_cache'
- 但 extended_cache.py 中实际定义的是 'technical_cache'
- 导入名称不匹配导致 ImportError

影响:
- 技术指标缓存功能无法使用
- 每次计算技术指标都会重新计算，影响性能

问题2: PredictionResult key_factors 验证错误
--------------------------------------------------------------------------------
位置: backend/services/prediction_engine.py 第479-484行
错误: 4 validation errors for PredictionResult
      key_factors.0.factor
      Input should be a valid number, unable to parse string as a number

原因:
- prediction_engine.py 中的默认 key_factors 格式为:
  {'factor': '技术面', 'weight': 35.0}
- 但 PredictionResult 模型定义为:
  key_factors: List[Dict[str, float]]  # [{factor: weight}]
- Pydantic 期望 Dict 的值是 float，但 'factor' 键的值是字符串

影响:
- 当模型未训练时，生成预测失败
- 返回默认预测时触发验证错误

================================================================================
修复方案
================================================================================

修复1: technical_indicators_cache 导入错误
--------------------------------------------------------------------------------
文件: backend/services/stock_service.py

修改内容:
1. 第468行 - 导入语句
   原代码: from services.extended_cache import technical_indicators_cache
   改为:   from services.extended_cache import technical_cache

2. 第470行 - 使用语句
   原代码: cached_indicators = await technical_indicators_cache.get(stock_code)
   改为:   cached_indicators = await technical_cache.get(stock_code)

3. 第530行 - 使用语句
   原代码: await technical_indicators_cache.set(result.dict(), stock_code)
   改为:   await technical_cache.set(result.dict(), stock_code)

修复2: PredictionResult key_factors 验证错误
--------------------------------------------------------------------------------
文件: backend/services/prediction_engine.py

修改内容:
第479-484行 - key_factors 默认值格式

原代码:
    key_factors = [
        {'factor': '技术面', 'weight': 35.0},
        {'factor': '基本面', 'weight': 25.0},
        {'factor': '板块轮动', 'weight': 25.0},
        {'factor': '国际市场', 'weight': 15.0}
    ]

改为:
    key_factors = [
        {'技术面': 35.0},
        {'基本面': 25.0},
        {'板块轮动': 25.0},
        {'国际市场': 15.0}
    ]

说明:
- 修改为符合 List[Dict[str, float]] 的格式
- 字典的键是特征名称，值是对应的权重（float类型）
