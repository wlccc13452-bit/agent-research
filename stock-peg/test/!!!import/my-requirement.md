# 原则：优先前端从SERVER本地数据库读取，网络API作为SERVER后台更新源

1. 前端请求 → 后端从数据库读取 → 立即返回；前端所有行情数据只加载本地数据，不再调用网络API
2. SERVER端数据缺失 → 触发后台更新任务 → WebSocket推送更新给前端
3. 避免阻塞前端,提升前端、后端速度相响应速度
