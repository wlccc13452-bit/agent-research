class StockRealtimeQuote(Base):
    """股票实时行情数据表（本地缓存）"""
    __tablename__ = "stock_realtime_quote"

    stock_code = Column(String(10), unique=True, index=True)

### 2. 后端API修改

**文件**: `backend/routers/stock.py`

#### 修改前（会调用网络API）
```python
# 优先数据库，没有就调用网络API
if not quote:
    quote = await stock_service.get_quote(stock_code)  # ❌ 调用网络API
    await quote_data_service.save_quote_to_db(db, quote)  # 保存到数据库
```

#### 修改后（只读数据库）
```python
# 只从数据库读取，没有数据就返回404
quote = await quote_data_service.get_quote_from_db(db, stock_code)

if quote:
    return quote  # ✅ 返回数据库中的数据
    
raise HTTPException(status_code=404, detail="数据库中未找到行情数据")  # ✅ 不调用网络API
```

### 3. 前端修改

**文件**: `frontend/src/pages/StockDetail.tsx`

#### 修改前
```typescript
const quote = await stocksApi.getQuote(stockCode);
// 获取失败就报错
```

#### 修改后
```typescript
try {
    const quote = await stocksApi.getQuote(stockCode);
    if (quote) {
        // ✅ 有数据就显示
        return quote;
    } else {
        // ✅ 没有数据就不加载
        return null;
    }
} catch (error) {
    if (is404) {
        // ✅ 数据库无数据，不报错
        return null;
    }
    throw error;
}
```

## 🔄 数据流程

### 新的数据流程

```
前端请求行情
    ↓
后端API: /api/stocks/quote/{code}
    ↓
从数据库读取
    ├─ 有数据 → 返回数据 ✅
    └─ 无数据 → 返回404（不调用网络API）❌
    ↓
前端处理
    ├─ 有数据 → 显示行情
    └─ 无数据 → 不显示行情（不报错）
```

### 关键变化

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| **数据来源** | 网络API + 数据库 | ✅ 仅数据库 |
| **无数据时** | 调用网络API | ✅ 返回404 |
| **前端处理** | 必须有数据 | ✅ 可接受无数据 |
| **网络请求** | 频繁调用 | ✅ 不调用 |

## 🎯 核心特性

### ✅ 严格数据库优先

**原则**: 只加载数据库中已保存的数据

**实现**:
1. 后端API只从数据库读取
2. 数据库无数据返回404（不调用网络API）
3. 前端可接受无数据情况

### ✅ 无网络依赖

**好处**:
- 不依赖外部API
- 不受网络状况影响
- 不触发API限制
- 响应速度极快（<100ms）

### ✅ 数据安全

**优势**:
- 数据本地存储
- 完全可控
- 可随时更新/删除
