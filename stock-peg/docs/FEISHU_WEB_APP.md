## 🎯 功能特性

### 1. 飞书环境自动检测
- ✅ 自动识别是否在飞书 App 中打开
- ✅ 显示当前运行环境（飞书/浏览器）
- ✅ SDK 自动加载和初始化

### 2. 股票查询
- ✅ 支持股票代码查询（如：600519）
- ✅ 支持股票名称查询（如：贵州茅台）
- ✅ 实时行情数据展示
- ✅ 搜索历史记录（本地存储）

### 3. 数据展示
- ✅ 股票基本信息（代码、名称）
- ✅ 实时价格和涨跌幅
- ✅ 开盘价、最高价、最低价、昨收
- ✅ 成交量、成交额

### 4. 移动端优化
- ✅ 响应式布局，适配手机屏幕
- ✅ 底部固定操作栏（仅飞书环境）
- ✅ 快捷搜索历史

### 卡片按钮跳转

在飞书卡片中点击"查看详细网页版"按钮，会跳转到此页面。

**卡片配置示例**:
```json
{
  "tag": "button",
  "text": {"tag": "plain_text", "content": "查看详细网页版"},
  "type": "default",
  "multi_url": {
    "url": "http://172.16.105.145:5173/feishu",
    "android_url": "http://172.16.105.145:5173/feishu",
    "ios_url": "http://172.16.105.145:5173/feishu",
    "pc_url": "http://172.16.105.145:5173/feishu"
  }
}
```

### URL 参数传递

支持通过 URL 参数直接跳转到指定股票：

```
http://172.16.105.145:5173/feishu?code=600519
```

## 🔧 技术实现

### 前端技术栈
- **React 19** - UI 框架
- **TypeScript** - 类型安全
- **React Router** - 路由管理
- **TanStack Query** - 数据获取和缓存
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库

### 飞书 JSSDK 集成
- **useFeishu Hook** - 自定义 Hook 封装
  - 环境检测
  - SDK 动态加载
  - 鉴权初始化
  - 窗口关闭

### 类型定义
- **feishu.d.ts** - TypeScript 类型声明
  - `LarkSDK` 接口
  - `window.lark` 类型扩展

## 🚀 开发指南

### 本地开发
```bash
# 启动前端服务
cd frontend
npm run dev

# 访问页面
http://localhost:5173/feishu
```

### 构建生产版本
```bash
cd frontend
npm run build
```

### 环境变量配置
确保 `backend/.env` 中配置了飞书应用凭证：
```env
FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
```

## 🐛 常见问题

### Q: 页面显示"浏览器"而不是"飞书环境"？
A: 确保在飞书 App 中打开页面，而不是浏览器。检查 User-Agent 是否包含 `lark` 或 `feishu`。

### Q: SDK 加载失败？
A: 
1. 检查网络连接
2. 确认飞书 JSSDK URL 可访问：https://open.larksuite.com/open/apis/jssdk/lark/h5_sdk.js
3. 查看浏览器控制台错误信息

### Q: 无法搜索股票？
A: 
1. 确认 backend 服务正在运行
2. 检查 API 端点配置
3. 查看网络请求是否成功

### Q: 点击"关闭"按钮无反应？
A: 
1. 确认在飞书环境中
2. 检查 SDK 是否加载成功
3. 查看控制台是否有权限错误
