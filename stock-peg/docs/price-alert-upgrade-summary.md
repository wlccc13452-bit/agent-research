# 价格提醒系统升级总结

## 升级概述

本次升级实现了飞书价格提醒系统的四个核心功能增强，从"被动响应"升级为"主动智能监控"。

## 升级内容

### 1. 交互升级：从"对话"转为"填表" ✅

**改进点：**
- 新增独立的上涨预警%和下跌预警%输入字段
- 参考价格、上涨预警%、下跌预警%字段均设置 `input_type="number"`，自动弹出数字键盘
- 遵循飞书 1.5.3 版本兼容性规范，input 组件的 label 直接写为字符串

**实现位置：**
- `backend/services/feishu_card_service.py` - `_create_monitor_setup_card()` 方法

**字段说明：**
- `stock_code`: 股票代码（必填）
- `ref_price`: 参考价格（可选，数字输入）
- `up_alert_pct`: 上涨预警百分比（可选，如 5.0 表示 5%）
- `down_alert_pct`: 下跌预警百分比（可选，如 3.0 表示 3%）
- `notes`: 备注（可选）

### 2. 逻辑闭环：回调响应与数据持久化 ✅

**改进点：**
- 表单提交后解析 `up_alert_pct` 和 `down_alert_pct` 字段
- 数据持久化到数据库，支持不对称预警（上涨5%、下跌3%）
- 函数末尾显式返回 `{"toast": "✅ 监控已开启"}`，消除 200672 错误

**实现位置：**
- `backend/services/feishu_long_connection_service.py`
  - 表单字段提取：Line 691-693
  - `_handle_monitor_task_submission()` 方法：完整重构

**数据流：**
1. 用户填写表单 → 提交
2. 解析 form_value 中的字段
3. 验证预警条件（至少设置一个）
4. 调用 `price_alert_manager.create_alert()` 持久化
5. 返回 Toast 确认消息

### 3. 主动推送：监控发现后的"弹窗"提醒 ✅

**改进点：**
- 预警卡片根据涨跌幅动态切换颜色（上涨=red，下跌=green）
- 核心价格对比：`[基准价: ¥15.00] → [实际价: ¥16.50] | 变化: 📈 +1.50元 (+10.00%)`
- 参考价格偏差显示：`🎯 参考价格: ¥16.00 (偏差: +0.50元 / +3.12%)`
- 触发条件清晰标注：`📈 上涨预警: +5.00% | 📉 下跌预警: -3.00%`

**实现位置：**
- `backend/services/price_alert_monitor_service.py` - `_create_alert_notification_card()` 方法

**颜色规则：**
- 上涨预警：红色主题 (`template: "red"`)
- 下跌预警：绿色主题 (`template: "green"`)
- 目标价达成：根据价格方向决定颜色

### 4. 自动化调度：确保监控永不掉线 ✅

**改进点：**
- 精准交易时段调度：仅在 A 股交易时间检查（9:30-11:30, 13:00-15:00）
- 高频检查：交易时段每 1 分钟执行一次 `check_all_alerts()`
- 状态管理：预警触发后自动设置 `is_triggered=1`，避免重复推送
- 持久化：触发时间、原因、通知状态全部记录到数据库

**实现位置：**
- `backend/services/scheduler.py` - 价格提醒监控调度配置（Line 118-142）
- `backend/services/price_alert_monitor_service.py`
  - `_get_active_alerts()`: 查询时过滤已触发预警 (`is_triggered == 0`)
  - `_trigger_alert_notification()`: 触发后设置状态 (`is_triggered = 1`)

**调度任务：**
1. 上午盘前半段 (9:30-10:59) - 每分钟检查
2. 上午盘后半段 (11:00-11:30) - 每分钟检查
3. 下午盘前半段 (13:00-14:59) - 每分钟检查
4. 下午盘最后检查 (15:00) - 整点检查

## 技术亮点

### 1. 类型安全
- 所有数字输入字段设置 `input_type="number"`
- 使用 `Decimal` 进行精确的金融计算

### 2. 用户体验
- 表单卡片一次性填写所有参数
- 数字键盘自动弹出
- 预警卡片视觉对比强烈

### 3. 性能优化
- 仅在交易时段运行监控
- 数据库查询过滤已触发预警
- 状态管理避免重复推送

### 4. 可扩展性
- 支持不对称预警（上涨5%、下跌3%）
- 参考价格可选，未设置时使用当前价格
- 备注字段支持用户自定义信息

## 使用示例

### 创建预警

1. 用户点击"创建价格监控"
2. 填写表单：
   - 股票代码：`000001`
   - 参考价格：`15.00`
   - 上涨预警%：`5.0`
   - 下跌预警%：`3.0`
   - 备注：`平安银行关键价位`
3. 点击"✅ 开启监控"
4. 系统返回：`✅ 监控已开启`

### 触发预警

场景：平安银行从 15.00 元涨到 15.75 元（涨幅 5%）

预警卡片内容：
```
📈 涨幅预警
平安银行 (000001)

📊 价格对比: [基准价: ¥15.00] → [实际价: ¥15.75] | 变化: 📈 +0.75元 (+5.00%)

🎯 参考价格: ¥15.00 (偏差: +0.75元 / +5.00%)

当前涨跌幅: 📈 +5.00%
触发条件: 涨幅已达到 +5.00%

监控阈值: 📈 上涨预警: +5.00% | 📉 下跌预警: -3.00%

🕐 触发时间: 2026-03-17 14:32:15
```
