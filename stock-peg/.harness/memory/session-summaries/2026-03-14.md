### 3. Market Breadth Visualization (Shanghai Index vs Up Ratio) ✅
**Requirement**: Add a small chart showing both Shanghai Index and up-ratio side by side.

**Implementation**:

#### New Backend APIs:
1. **`GET /api/market/index-sh-history`**
   - Returns Shanghai Index (000001) closing prices
   - Parameter: `days` (default: 30, max: 365)

2. **`GET /api/market/market-sentiment/history`**
   - Returns historical market sentiment data
   - Parameter: `days` (default: 30, max: 365)
   - Returns: date, total_count, up_count, down_count, market_breadth, limit_up, limit_down
