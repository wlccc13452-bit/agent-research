# 百分比格式化规则说明

## 📋 格式化规则

### 规则优先级（从高到低）

1. **超过±5%** → 不显示小数
   - 示例：`+20.0%` → `+20%`
   - 示例：`-10.5%` → `-10%`

2. **小数点后第一位是0** → 不显示小数
   - 示例：`+5.0%` → `+5%`
   - 示例：`+10.0%` → `+10%`
   - 示例：`+15.0%` → `+15%`

3. **1%-5%且第一位小数不是0** → 显示一位小数
   - 示例：`+2.5%` → `+2.5%`
   - 示例：`+3.8%` → `+3.8%`
   - 示例：`-1.2%` → `-1.2%`

4. **其他情况** → 显示一位小数
   - 示例：`+0.5%` → `+0.5%`
   - 示例：`+0.8%` → `+0.8%`

## 📊 完整示例对照表

| 原始值 | 格式化后 | 应用规则 |
|--------|---------|---------|
| `+20.0` | `+20%` | 规则1：超过5% |
| `+15.0` | `+15%` | 规则2：第一位小数是0 |
| `+10.5` | `+10%` | 规则1：超过5% |
| `+5.0` | `+5%` | 规则2：第一位小数是0 |
| `+3.8` | `+3.8%` | 规则3：1-5%且第一位小数不是0 |
| `+2.5` | `+2.5%` | 规则3：1-5%且第一位小数不是0 |
| `+1.2` | `+1.2%` | 规则3：1-5%且第一位小数不是0 |
| `+0.8` | `+0.8%` | 规则4：其他情况 |
| `+0.5` | `+0.5%` | 规则4：其他情况 |
| `-10.0` | `-10%` | 规则2：第一位小数是0 |
| `-20.5` | `-20%` | 规则1：超过5% |
| `-3.8` | `-3.8%` | 规则3：1-5%且第一位小数不是0 |

## 🔧 实现逻辑

```typescript
const formatPercent = (value: number | null | undefined): string => {
  if (value === null || value === undefined) return '--';
  
  const abs = Math.abs(value);
  
  // 规则1: 超过5%不显示小数
  if (abs > 5) {
    return (value > 0 ? '+' : '') + value.toFixed(0) + '%';
  }
  
  // 规则2: 小数点后第一位是0不显示小数
  const firstDecimal = Math.floor((abs % 1) * 10);
  if (firstDecimal === 0) {
    return (value > 0 ? '+' : '') + value.toFixed(0) + '%';
  }
  
  // 规则3: 1%-5%且第一位小数不是0，显示一位小数
  return (value > 0 ? '+' : '') + value.toFixed(1) + '%';
};
```

## 📝 应用场景

### 自持股票面板 (HoldingsPanel)

```typescript
// 上涨控制
{formatPercent(stock.change_up_pct)}
// 示例: change_up_pct = 20 → "+20%"

// 下跌控制（注意：下跌控制本身是正数，显示时加负号）
{formatPercent(stock.change_down_pct ? -stock.change_down_pct : null)}
// 示例: change_down_pct = 10 → "-10%"

// 目标价涨幅
{formatPercent(targetChangePct)}
// 示例: targetChangePct = 2.5 → "+2.5%"
```

### 动态关注股票面板 (LeftPanel)

```typescript
// 同上
{formatPercent(stock.change_up_pct)}
{formatPercent(stock.change_down_pct ? -stock.change_down_pct : null)}
{formatPercent(targetChangePct)}
```
