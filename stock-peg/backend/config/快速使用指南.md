# 控制台输出配置 - 快速使用指南

## 🎯 目标

减少server控制台的冗余输出，让飞书机器人日志更清晰可见。

## 📁 配置文件位置

```
backend/config/console_output.ini
```

## ⚡ 快速配置

### 场景1：生产环境（推荐）

**特点**：只显示重要信息和飞书日志

```ini
[console_output]
# 总开关
enable_console_output = true

# 必须开启（重要！）
enable_errors = true
enable_warnings = true
enable_startup_info = true
enable_system_status = true

# 飞书机器人（默认已开启）
enable_feishu_bot = true
enable_feishu_connection = true
enable_feishu_chat = true

# 其他全部设为 false（减少噪音）
enable_stock_data = false
enable_market_index = false
enable_kline_data = false
...（其他false）
```

**效果**：
- ✅ 启动信息可见
- ✅ 错误和警告可见
- ✅ 飞书机器人日志可见
- ❌ 股票数据日志隐藏
- ❌ 更新进度日志隐藏
- ❌ WebSocket日志隐藏

---

### 场景2：调试股票数据

```ini
[console_output]
enable_console_output = true
enable_errors = true
enable_warnings = true
enable_debug = true

# 开启需要调试的数据类型
enable_stock_data = true
enable_kline_data = true
enable_update_progress = true

# 保持飞书日志
enable_feishu_bot = true
```

**效果**：可以看到股票数据的详细日志

---

### 场景3：完全静默

```ini
[console_output]
enable_console_output = false
enable_errors = false
```

**效果**：控制台几乎无输出

---

## 🔧 使用API管理配置

### 查看当前配置

```bash
GET http://localhost:8000/api/config/console-output
```

### 查看配置摘要

```bash
GET http://localhost:8000/api/config/console-output/summary
```

返回示例：
```json
{
  "console_output_enabled": true,
  "errors_enabled": true,
  "warnings_enabled": true,
  "feishu_enabled": true,
  "data_outputs_enabled": 2,
  "data_outputs_total": 14,
  "status": "quiet",
  "recommendation": "配置简洁，适合生产环境"
}
```

### 修改单个配置项

```bash
POST http://localhost:8000/api/config/console-output/update
Content-Type: application/json

{
  "config_key": "enable_stock_data",
  "value": true
}
```

### 应用预设配置

```bash
# 安静模式（生产环境推荐）
POST http://localhost:8000/api/config/preset/quiet

# 正常模式（开发环境）
POST http://localhost:8000/api/config/preset/normal

# 调试模式
POST http://localhost:8000/api/config/preset/debug

# 仅飞书日志
POST http://localhost:8000/api/config/preset/feishu-only
```

**注意**：API修改后需要重启服务才能生效！

## 📊 配置项说明

### 飞书机器人专用（默认开启）

| 配置项 | 默认值 | 说明 |
|-------|--------|------|
| `enable_feishu_bot` | ✅ true | 飞书消息收发 |
| `enable_feishu_connection` | ✅ true | 飞书长连接状态 |
| `enable_feishu_chat` | ✅ true | 飞书对话历史 |

**建议**：保持开启，方便监控机器人运行状态

### 错误和警告（建议开启）

| 配置项 | 默认值 | 说明 |
|-------|--------|------|
| `enable_errors` | ✅ true | 错误日志（重要！） |
| `enable_warnings` | ✅ true | 警告日志（重要！） |
| `enable_debug` | ❌ false | 调试日志 |

**建议**：生产环境保持 errors 和 warnings 开启

### 数据类输出（默认关闭）

| 配置项 | 默认值 | 说明 |
|-------|--------|------|
| `enable_stock_data` | ❌ false | 股票数据读写 |
| `enable_market_index` | ❌ false | 市场指数 |
| `enable_kline_data` | ❌ false | K线数据 |
| `enable_financial_data` | ❌ false | 财务数据 |
| `enable_us_market` | ❌ false | 美股市场 |

**建议**：调试数据问题时开启，日常关闭

### 功能类输出（默认关闭）

| 配置项 | 默认值 | 说明 |
|-------|--------|------|
| `enable_update_progress` | ❌ false | 数据更新进度 |
| `enable_websocket` | ❌ false | WebSocket连接 |
| `enable_cache` | ❌ false | 缓存操作 |

## ⚠️ 注意事项

1. **错误日志建议保持开启**
   ```ini
   enable_errors = true
   enable_warnings = true
   ```
   生产环境不建议关闭错误和警告日志

2. **飞书日志默认已开启**
   如无特殊需求，保持飞书相关配置为 true

3. **配置修改后需重启**
   修改INI文件后，必须重启服务才能生效

4. **API修改也需重启**
   通过API修改配置后，同样需要重启服务

## 🔍 故障排查

### 问题1：看不到飞书日志

**检查**：
```bash
GET http://localhost:8000/api/config/console-output
```

确认：
- `enable_console_output = true`
- `enable_feishu_bot = true`
- `enable_feishu_connection = true`

### 问题2：错误日志被隐藏

**解决**：
```ini
enable_errors = true
enable_warnings = true
```

### 问题3：配置没生效

**原因**：未重启服务

**解决**：Ctrl+C 停止服务，重新运行 `python main.py`
