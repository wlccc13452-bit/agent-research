第一步：注册 NVIDIA 账号
打开官网：https://build.nvidia.com

进入 API Keys 管理页面，点击右上角的 「+ Create API Key」 按钮
在弹出的对话框中：
 Name：随便填一个名字，比如 my-free-key

Expiration：有效期 —— 这里可以选最长 12 个月（1 年）
点击 「Create Key」 确认
重要提示： API Key 创建成功后只显示一次，页面上会出现一串 nvapi- 开头的字符串，务必立刻复制保存，关闭弹窗后就看不到完整 Key 了
第四步：找到你想用的模型 ID
回到 https://build.nvidia.com/models，这里有 100+ 个可用模型
点进任意一个模型页面（比如 minimax-m2.7），右侧代码示例里会显示该模型的 Model ID

NVIDIA 托管了许多针对特定任务优化的模型，你可以根据需求选择：

| **任务类型**        | **推荐模型**                        | **特点**                         |
| ------------------------- | ----------------------------------------- | -------------------------------------- |
| **通用/逻辑推理**   | `meta/llama-3.3-70b-instruct`           | 开源界最强梯队，逻辑能力极佳。         |
| **中文增强/多语言** | `qwen/qwen2.5-72b-instruct`             | 阿里通义千问系列，中文处理能力顶尖。   |
| **极速响应**        | `meta/llama-3.1-8b-instruct`            | 参数量小，推理速度极快，适合简单对话。 |
| **长文本处理**      | `mistralai/mixtral-8x22b-instruct-v0.1` | 混合专家模型，擅长处理复杂的上下文。   |
| **视觉理解**        | `nvidia/neva-22b`                       | 支持图像输入和多模态理解。             |
