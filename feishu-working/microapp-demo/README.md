## ## 背景

​	本项目用于帮助小程序开发者快速入门，提供各种小程序`组件`和`API`示例代码。


## 安装

​	方式一：使用[飞书开发工具](https://open.feishu.cn/document/uYjL24iN/uATMuATMuATM)导入本项目即可使用。

​	方式二：使用Visual Studio Code导入本项目，结合[Feishu Extension](https://open.feishu.cn/document/uYjL24iN/uQzN3QjL0czN04CN3cDN)即可使用。


## 使用

​	本项目提供`功能模块`-`组件`的基本样式，开发者可根据自身需求自定义组件样式。`功能模块`-`API`将演示小程序接口的能力，开发者可参考使用。


## 功能模块

	### 组件

​	（1）基础内容：基础内容组件的使用示例

​	（2）视图容器：视图容器组件的使用示例

​	（3）表单：表单相关组件的使用示例

​	（4）导航：导航栏组件的使用示例

​	（5）媒体组件：媒体相关组件的使用示例

​	（6）其他：其他组件的使用示例

### API

​	（1）开放接口：飞书对外开放接口的使用示例

​	（2）网络：网络相关接口的使用示例

​	（3）媒体：媒体相关接口的使用示例

​	（4）数据缓存：数据缓存相关操作的使用示例

​	（5）位置：位置相关接口的使用使用示例

​	（6）设备：设备相关接口的使用示例

​	（7）界面：界面操作相关接口的使用示例

​	（8）导航：导航相关接口的使用示例
